# Restaurant-AI-System
AI-powered system that helps restaurants manage dishes, ingredients, and allergies, enabling staff to quickly accommodate guest needs and improve training with accurate, consistent knowledge of the restaurant offerings.

## Current Status

### Completed
- Developed an initial web application interface
- Deployed the application on AWS using Amplify for hosting and infrastructure management

### Next Steps
- **Authentication System**: Implement a secure sign-in system to enable restaurant accounts
- **MCP Server & LLM Integration**: Develop a Model Context Protocol (MCP) server with custom tools to enable seamless integration with advanced language models, providing intelligent automation capabilities for restaurant operations
